# Multi-threading-and-Synchronization
## GROUP PROJECT CSCI 3431: Operating System(Winter 2022)

The main task in this project is to design and develop a multi-threading application which involves synchronization. The program implements Floyd-Warshall (FW) All-Pairs-Shortest-Path algorithm in a multi-threaded fashion along with enforcing the readers-writers problem.
